# Практическое занятие 4. Запросы к данным (SELECT)

**Продолжительность:** 2 академических часа (90 минут)

**Тип:** практическое занятие

**Цель занятия:** сформировать устойчивые навыки выборки данных из одной и нескольких таблиц; фильтрации (WHERE); сортировки (ORDER BY); ограничения выборки (LIMIT); работы с условиями (IN, BETWEEN, LIKE, IS NULL); соединения таблиц (JOIN)

## Исходная структура БД

```sql
CREATE TABLE groups (
    id SERIAL PRIMARY KEY,
    name TEXT NOT NULL UNIQUE
);

CREATE TABLE students (
    id SERIAL PRIMARY KEY,
    full_name TEXT NOT NULL,
    birth_date DATE,
    group_id INT REFERENCES groups(id),
    scholarship NUMERIC
);

CREATE TABLE teachers (
    id SERIAL PRIMARY KEY,
    full_name TEXT NOT NULL,
    department TEXT
);
```

> Перед началом занятия таблицы должны быть заполнены тестовыми данными (10–20 строк).
